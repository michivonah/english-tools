English Tools

Hello!

Herzlich Willkommen auf english.math-tools.app. Das wird eine Sammlung von diversen Mathematik Tools. Es soll dazu dienen dir den Alltag zu erleichtern.

Links siehst du eine Liste mit allen bisher vorhanden Tools. Klicke darauf um es zu öffnen! Die Liste kann ausserdem mit dem kleinen Icon unten links vergrössert und verkleinert werden.

Was noch alles für dieses Tool geplant ist, siehst du auf dem GitHub Repo zu dieser Website.

Wenn dir ein Tool fehlt, dann erstelle ein Issue bei GitHub.

Wenn du mehr über mich erfahren willst, besuche meine Website auf michivonah.ch.

Und nun, viel Spass mit den Tools. ;-)

Flächen & Volumen

Fülle die Felder aus, um die entsprechende Fläche oder das Volumen zu berechnen. Funktioniert für Quadrate, Rechtecke und Quader.

Hinweis zum Datenschutz

Die auf dieser Website vom Benutzer angegebenen Daten werden nicht gespeichert und nur lokal auf dem Gerät zur Berechnung der gewünschten Werte verwendet. Dies kann in der Datenschutzerklärung von GitHub nachgelesen werden. Ausserdem werden die Aufrufe mit Ackee Analytics getrackt.
